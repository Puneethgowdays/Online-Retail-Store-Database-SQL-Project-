# Online Retail Store Database (SQL Project)

This project demonstrates a complete end-to-end SQL solution similar to real-world retail business systems. It includes database design, relationships, stored procedures, triggers, views, and analytical reporting queries.

## Features Implemented

### 1. **Database Design**
- Customers
- Products
- Orders
- Order Items
- Order Log (audit table)

### 2. **Core SQL Concepts**
- Table creation with keys
- Data insertion
- Joins (INNER JOIN)
- Aggregations (SUM, GROUP BY)
- Constraints and relationships

### 3. **Advanced SQL**
- **View**: Daily sales summary  
- **Stored Procedure**: Fetch customer order history  
- **Trigger**: Automatically logs every new order  
- **Analytical Reports**: Top customers, best-selling products  
 
## How to Use
Run the script `retail_store.sql` in MySQL Workbench / SQL Server / PostgreSQL
to create all tables and test queries.

This project represents hands-on working knowledge of SQL for application
development and data analysis.
